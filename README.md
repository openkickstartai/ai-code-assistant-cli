## Installation

```bash

```

## Usage

```bash
# Get code completion
ai-assist complete --file my_script.py --line 25

# Refactor code
ai-assist refactor --file my_script.py --function my_function

# Generate documentation
ai-assist docs --file my_script.py

# Git commit assistance
ai-assist commit --staged
```

- Python 3.8+
